LocationServices for Windows Phone

a Reactive Extensions & RestSharp powered service layer for Windows Phone wrapping the data from Stockholm city, Göteborg city, Örebro city, Bing Locations, Bing Translation, CommuteGreener, Flickr, Twitter, Nokia Places, Google Places & Instagram.

Download this project as a .zip file Download this project as a tar.gz file

Welcome

This framework is created to speed up your development time and to keep a structured, single approach to fetching data and converting it to observables.

Sources

In the current state the framework has 11 data sources included:

  • Bing Locations & Translation
  • Commute Greener
  • Flickr
  • Google Places
  • Göteborg city
  • Instagram
  • Nokia
  • Örebro city
  • Stockholm city
  • Twitter
  • Eniro

Cachable

Yeah, it has built-in caching support.
